## 前言
  我之前是学SpringMVC的，后面听同学说SpringBoot挺好用，极力推荐我学这个鬼。一开始，在网上找SpringBoot的学习资料，他们博文写得不是说不好，而是不太详细。我就在想我要自己写一篇尽可能详细的文章出来。

## 技术栈
- Spring
- Spring Boot
- MyBatis
- MongoDB
- MySQL

## 设计模式

MVC

## 功能
- 注册（用户完成注册后是默认未激活的，程序有个定时器在检测没有激活的用户，然后发一次邮件提醒用户激活）
- 登录
- 发帖（帖子存在MongoDB）
- 评论 
- 其他功能正在添加中...
